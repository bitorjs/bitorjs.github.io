使用 bitorjs 进行vue 项目开发 第一步安装bitorjs-cli 工具包 > npm i -g bitorjs-cli 第二步 创建新的bitor项目，并选择项目模板（-r 使用 React, -v 使用 Vue, 默认为 Vue） bitor n [project-name] 第三步 进入项目文件 , cd project-name 进行vue的开发 npm i 安装项目依赖 npm start 启动本地开发 设置启动页面 start.vue, 内页使用 webview-container 组件占位（所以路由都会导入动态渲染到 此组件上） 开发其它 Vue 页面组件 设置路由： 本项目不使用传统 Vue 中路由系统，而是使用 controller 类进行页面路由的控制。。这也是bitor框架的特点 至此，bitorjs 框架下的 Vue 项目的开发已在进行中 其它项目配置说明： 1、自定义文件监听 2、自动化注入（模板框架仅自动化注入了 Controller 类文件，其它的如 view 组件并没有做处理，公共的全局Vue组件也已自动化处理） 基本不改变用户的项目开发的习惯，仅把 vue 项目中路由拿了出来，在此做数据处理的中间层，根据需要，你也可以添加更多的数据处理层，比如 Service 层， 当然不是为了技术而技术，而是更方便让不太喜欢把路由作为节点的开发者使用。 本框架其它知识点 1、路由规则 2、路由顺序 3、参数传递 4、请求链路
